## How to Use

- **In Canva:** Copy an "Open" link → In your Canva slide, go to Apps → Embeds → paste the link
- **In a browser:** Click "Open" to view the interactive version
- **Share with participants:** Send any link directly; all work on any device with a browser

## URL Pattern for Files in This Repo
